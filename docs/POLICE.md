# Vision

The police system makes RideVerse feel realistic and challenging.

Police should protect the city, enforce laws, investigate crimes, chase criminals, arrest wanted players, and respond intelligently using AI.

The police should never feel predictable.

Police decisions are server-authoritative.

Never trust client-side crime data.

Always synchronize important police events.

Optimize AI for large multiplayer servers.
